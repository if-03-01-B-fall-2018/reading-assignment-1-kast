World Wide Web

HTML,URL,HTTP characterize the World Wide Web.
HTML Sites and Pictures get linked together with URL and Hyperlinks.
URL is a web Adress with URI(Identifier).

Webserver

The Webserver is a server, the documents to clients such as Web browser transmits. A web server is the computer with web server software or only the web server software itself.
For each required file, the web browser must send its own request to the web server.
Web content is textual, visual or aural content.

Web Client

Der Web Client is an application program based on the client-server model.
The data processing and evaluation mainly takes place on a remote web server.
Web Client and Web Browser are identical
